Proyecto NAP

Sistema de Gestión de Puntos de Venta COPEC (POS)

Desarrollo del software de punto de venta POS para las estaciones de servicio de COPEC, compatible con terminales Verifone VX520.

  • Protocolo ISO 8583
  • Protocolo SPDH
  • Tarjetas Chip y Contactless
  • Lenguage C
  • Integración con DTE (Facturación Electrónica)
  • Clientee: Copec
  • Fecha: Enero 2014 - Julio 2014
  • Servicio: Desarrollo empresarial

Contact Center

Levantamiento, análisis, diseño e implementación de un clúster Big Data para procesamiento de datos masivos e históricos, derivados de interacciones telefónicas y multimedia de los ejecutivos del Call Center.

  • Cliente: Entel
  • Fecha: Enero 2015 - Mayo 2015
  • Servicio: Desarrollo Web Empresarial

Firma Electrónica de Documentos

Implementación de un sistema que permite de manera amigable, administrar centralizadamente las definiciones y lógica de generación de documentos que requerieren ser firmados electrónicamente.

Implementado utilizando tecnologia Oracle Service Bus (OSB), Enterprise Java Bean (EJB), Servicios Web (SOAP).

  • Cliente: CMR Falabella
  • Fecha: Diciembre 2014 - Abril 2015
  • Servicio: Desarrollo Web Empresarial - Servicios SOAP en OSB

Sistema de Gestión Productiva

Implementación de un sistema de “ Administración y Generación de Plantillas de Documentos”, además de realizar la integración de este software en el “Sistema de Gestión de Producción de Tarjetas e Impresión” de Vigatec.

  • Cliente: Vigatec
  • Fecha: Junio 2015 - Agosto 2015
  • Servicio: Desarrollo Empresarial

Patpass

Mejora del mecanismo de integración con comercios que actualmente ofrece PatPass, con el objetivo de manejar la inscripción de los tarjetahabientes, de manera de hacer el proceso más seguro, en línea y siguiendo los estándares corporativos.

  • Cliente: Transbank
  • Fecha: Julio 2014 - Septiembre 2014
  • Servicio: Desarrollo Empresarial

Venta Transparente

“Venta Transparente”, proyecto de pantallas auxiliares que permite a los clientes visualizar datos de sus transacciones, se integra con los sistemas corporativos.

  • Cliente: CMR Falabella
  • Fecha: Enero 2015 - Abril 2015
  • Servicio: Desarrollo Web Empresarial

SFCall

Construcción de un Call Center para la venta de seguros, integrándose con la plataforma comercial de Seguros Falabella y el software Genesys.

  • Cliente: Seguros Falabella
  • Fecha: Enero 2014 - Julio 2014
  • Servicio: Desarrollo Web Empresarial
